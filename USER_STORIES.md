# User Stories

## Like Feature
**As a** user  
**I want to** swipe right or tap the like button on a profile  
**So that** I can express interest in that person and potentially match with them

### Acceptance Criteria
- User can swipe right on a card to like the profile
- User can tap the heart button to like the profile
- Liked profile is removed from the deck
- Like action is visually confirmed with animation
- Profile is added to the "liked" list

## Reject Feature
**As a** user  
**I want to** swipe left or tap the reject button on a profile  
**So that** I can dismiss profiles I'm not interested in and see new ones

### Acceptance Criteria
- User can swipe left on a card to reject the profile
- User can tap the X button to reject the profile
- Rejected profile is removed from the deck
- Reject action is visually confirmed with animation
- Profile is added to the "rejected" list

## Super Like Feature
**As a** user  
**I want to** swipe up or tap the super like button on a profile  
**So that** I can show extra interest and stand out from regular likes

### Acceptance Criteria
- User can swipe up on a card to super like the profile
- User can tap the star button to super like the profile
- Super liked profile is removed from the deck
- Super like action is visually confirmed with distinct animation
- Profile is added to the "super liked" list with priority status
- User has limited super likes per day (e.g., 1 per 24 hours)
